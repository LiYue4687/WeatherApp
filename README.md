# WeatherApp
Develop a weather app based on compose. Contains third-party libraries such as Room and Retrofit.

# FrameWork
```
.
├── java
│   └── com
│       └── example
│           └── weatherapp
│               ├── data
│               │   ├── retrofit
│               │   │   └── entity
│               │   └── room
│               │       ├── dao
│               │       ├── database
│               │       ├── entity
│               │       └── repository
│               └── ui
│                   ├── add
│                   ├── theme
│                   └── weather
│                       ├── util
│                       └── widget
└── res
    ├── drawable
    ├── mipmap-anydpi
    ├── mipmap-hdpi
    ├── mipmap-mdpi
    ├── mipmap-xhdpi
    ├── mipmap-xxhdpi
    ├── mipmap-xxxhdpi
    ├── values
    └── xml
```
